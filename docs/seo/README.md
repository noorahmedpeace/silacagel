# DryGelWorld SEO Execution Pack

This folder contains the working SEO, authority, indexing, conversion, and content execution plans for DryGelWorld.

## Priority Order

1. **GSC and indexation**
   - Start with `GSC-GROWTH-PLAN.md`.
   - Use it to diagnose indexing buckets, sitemap consistency, cannibalization, and crawl priority.

2. **Internal authority flow**
   - Use `INTERNAL-LINKING-STRATEGY.md`.
   - Highest ROI items: export-market related links, orphan landing-page links, `/industries` hub, breadcrumbs, footer market coverage.

3. **International SEO**
   - Use `INTERNATIONAL-SEO.md`.
   - Focus: enrich existing export pages, add Africa market pages, clarify country intent, improve geo schema.

4. **Conversion and lead generation**
   - Use `LEADGEN-OPTIMIZATION.md`.
   - First requirement: verify RFQ email delivery through Resend and add buyer acknowledgments before scaling traffic.

5. **Core Web Vitals**
   - Use `CWV-OPTIMIZATION.md`.
   - Focus first on hero image loading, unused JS dependencies, video poster/compression, and long-cache headers.

6. **Content expansion**
   - Use `CONTENT-PACK.md`.
   - Prioritize commercial pages and guides that improve money-page depth and reduce thin/templated content risk.

7. **Backlinks and authority**
   - Use `BACKLINK-STRATEGY.md`.
   - Build after the technical, indexing, schema, internal-linking, and content-depth foundation is stable.

8. **SilicaGelPK authority bridge**
   - Use `AUTHORITY-FLOW-STRATEGY.md`.
   - Important: confirm SilicaGelPK is clean and secure before adding or expanding links.

9. **SilicaGelPK content draft**
   - `BLOG-SilicaGelPK-container-rain.md` is a WordPress-ready draft for SilicaGelPK.
   - Keep outbound links limited, contextual, and non-spammy.

## Guardrails

- Do not claim FDA, REACH, Halal, GMP, JEDEC, USP, or food-grade certification unless the business has proof.
- Safe proof language: ISO 9001:2015, SDS, COA, and DMF-free statement.
- Avoid exact-match anchor repetition.
- Do not build sitewide/footer backlinks between owned sites.
- Prioritize buyer-intent pages and pages already getting impressions.
- Keep desktop UI stable unless a critical issue requires a responsive fix.
